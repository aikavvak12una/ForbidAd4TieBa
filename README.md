# ForbidAd4TieBa

一个轻量级的开源 Xposed 模块，用于去除百度贴吧的冗余广告并净化页面体验。  
**设置入口：** 贴吧「我的」页面 -> 长按右上角「设置」图标

## 功能特性

* **核心框架**
  * 支持动态反混淆
* **广告拦截**
  * 屏蔽开屏广告
  * 屏蔽信息流广告（首页推荐列表、App 推广）
  * 屏蔽评论区广告
* **内容屏蔽**
  * 屏蔽信息流视频帖子
  * 屏蔽首页直播卡片
  * 屏蔽信息流顶部进吧入口 
  * 屏蔽帖子详情页底部进吧横幅
* **界面精简**
  * 首页 Tab 精简（仅保留“推荐”，去除游戏图标）
  * 底部 Tab 精简（去除“小卖部”）
  * 进吧页面精简（去除顶部推荐列表）
* **扩展功能**
  * 启动应用自动签到
  * 禁止自动刷新
  * 信息流预加载
