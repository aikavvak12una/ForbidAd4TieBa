# ForbidAd4TieBa

一个轻量级的开源 Xposed 模块，用于去除百度贴吧的各种多余广告并净化页面体验。

## 功能特性
- **开屏广告去除**
- **信息流广告去除**：去除首页推荐列表的广告卡片及App推广
- **评论区广告去除**：去除评论区的广告卡片及App推广
- **直播内容屏蔽**：去除首页直播卡片
- **推荐进吧卡片屏蔽**：去除首页推荐进吧卡片
- **首页Tab精简**：仅保留关注和推荐Tab
- **底部Tab精简**：去除小卖部Tab
- **模块设置面板**：长按设置图标，可自定义开关各项功能

## 兼容性
- 贴吧客户端版本兼容测试：`22.2.1.1` `22.3.1.0`
- 其它版本自行测试
